# `append(second, first)`

Appends two lists together.

## Alternatives

* `append(second)(first)`

## Arguments

* `second (Array)`: The list to append.
* `first (Array)`: The list on which to append.

## Returns

* `(Array)`: The appended list.

## Example

```javascript
const second = [2, 3];
const first = [0, 1];

append(second, first);
// => [0, 1, 2, 3]
```
